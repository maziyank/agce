Understanding the world full of risks

Natural disasters are phenomena that can occur at any time, anywhere and at any time, causing risks or dangers to human life, both property losses and human casualties. There are many type of natural disaster like flood, storm, earthquake etc. Hence, knowing our natural conditions in which we live is considered important. Understanding disaster also can be used to support the decision-making process regarding prevention and preparedness for future disasters.

Using the Emergency Events Database (EM-DAT) published by the Center for Research on the Epidemiology of Disasters (CRED) we aim to visualize the disaster profile for each country from 2000 to 2021. However, you can filter by the year the disaster occurred using the slider below.

Through the map beside we can see that the occurrence of disasters in each country is different. Some countries often experience disasters and suffer substantial loss of life and infrastructures. The color difference reflect the value of certain variable for each country which can be the number of disaster events, the number of death tools, the number of people affected and financial damage losses. We can select color encoding for the map via the buttons below.

Furthermore, we can also see detailed disaster information for each country. A dialog will appear if we click on a country on the map. There we can find out many things such as what types of disasters often occur and how each type of disaster has a very detrimental negative impact.
